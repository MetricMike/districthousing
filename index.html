The Purpose

Accelerating the affordable housing application process.

Centralized Application

Application forms for affordable housing are not standard. District Housing allows social workers to automatically fill out PDFs for their customers.

Electronic Applications

As building forms change, District Housing allows social workers to add new application forms to the system.

Automated Entry

Pen and paper shouldn't be the norm. District Housing allows social workers to automatically fill out applications for multiple buildings.

The Problem

It shouldn't be this hard.

For the past year, the District Housing team has been working with a local non-profit, Bread for the City, to streamline the process of applying for affordable housing in the district. Currently, if you want to apply for Section 8 housing in DC, you need to fill out a paper application for each building you want to apply for. There are over 100 buildings in DC, each with their own paper application form. None of the forms are the same and none of them are online. According to Bread for the City, each person they serve applies for 4-6 buildings, spending approximately 15-20 minutes filling out each paper application. Even then they are not guaranteed housing.

Solution

Finding a place to call home.

To address this problem, Code for DC is building an online form that, when filled out, automatically maps the form input to PDF versions of each building's original application form. Most buildings are unwilling to use a different application form, so providing an application in the same format as what they are accustomed to is important to them.

If we can demonstrate the benefits of having an common application form for affordable housing, our dream would be that an official common application is created and our project is made obsolete.
